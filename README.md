# Lume Boutique - Vercel

Versão preparada para publicação na Vercel.

## Publicar

1. Importe esta pasta em um repositório Git ou envie o projeto pela Vercel.
2. Use o framework `Next.js`.
3. Mantenha o comando de build padrão: `npm run build`.
4. Opcionalmente, crie a variável `NEXT_PUBLIC_SITE_URL` com o endereço final do site.

Esta versão mantém a vitrine, catálogo, filtros, produtos e carrinho. O painel administrativo baseado em Cloudflare D1 não faz parte desta cópia.
